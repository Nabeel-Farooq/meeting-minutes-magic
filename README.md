# 📋 Meeting Notes App

A simple and efficient Meeting Notes application built using modern web technologies like **Vue.js** and **Supabase**. This app helps teams capture, organize, and manage meeting discussions in a structured way.

---

## 🚀 Features

- 📝 Create and manage meeting notes  
- 📅 Organize notes by date or meeting  
- 🔐 User authentication (Supabase Auth)  
- ☁️ Real-time database syncing  
- ⚡ Fast and responsive UI with Vue  

---

## 🛠️ Tech Stack

- **Frontend:** Vue.js  
- **Backend / Database:** Supabase  
- **Authentication:** Supabase Auth  
- **Styling:** CSS
